# 提案書ビルダー — 完全仕様書

## 概要

Web制作の提案書をブラウザ上で作成・管理するWebアプリケーション。フォームに案件情報を入力すると、プロフェッショナルな提案書がリアルタイムでプレビュー表示され、PDF出力できる。

**技術スタック（予定）:** Next.js (App Router) + React + Tailwind CSS  
**現在のプロトタイプ:** React (JSX) 単一ファイル — `proposal-builder.jsx`

---

## 画面構成

### ヘッダー
- アプリ名「提案書ビルダー」＋ロゴアイコン（SVG）
- ヘッダーカラーは選択中のテーマカラーに連動
- 右側ボタン群：
  - **金額表示/非表示トグル** — ON/OFFでプレビュー・PDF内の金額表示を切り替え。ONの時はボタンがハイライト
  - **保存** — 案件の保存・読み込みモーダルを開く
  - **PDF出力** — プレビュー内容を新規ウィンドウで印刷/PDF保存

### タブナビゲーション（6タブ）
すべてのタブにSVGアイコン付き。選択中のタブはテーマカラーのアンダーライン。

1. **基本情報**（clipboard）
2. **ページ構成**（folder）
3. **見積もり**（coins）
4. **スケジュール**（calendar）
5. **契約条件**（fileText）
6. **プレビュー**（eye）

### メイン領域
- タブ1〜5: 左側＝入力フォーム、右側＝縮小ライブプレビュー（68%スケール）
- タブ6（プレビュー）: 中央にフルサイズプレビュー表示

## 機能詳細

### 1. 基本情報タブ

#### 業種テンプレート
- 起動時にモーダルで業種選択を表示
- 8業種のプリセット：白紙、企業・コーポレート、飲食店・カフェ、美容院・サロン、クリニック・医療、士業、建設・リフォーム、採用サイト
- 各テンプレートに設定済み：サイト種別、案件概要、制作方針、実装機能、ページ構成、見積もり項目・金額、スケジュール
- 選択後もすべての項目を自由に編集可能
- 基本情報タブ内のボタンからいつでも再選択可能
- モーダルは外側クリックで閉じる

#### テーマカラー
- 18色のカラーパレットを色相順に配置
- 色順：クリムゾン → コーラル → ブラウン → ゴールド → オリーブ → フォレスト → ミント → ティール → オーシャン → スチール → ネイビー → スレート → インディゴ → ロイヤル → プラム → ワイン → ローズ → チャコール
- 選択するとアプリ全体（ヘッダー、タブ、ボタン、プレビュー）の色が即座に変更
- ガントチャート・フロー図の色はプライマリカラーから7段階のグラデーションを自動生成（`shades()` 関数）
- 選択中のカラーは少し拡大表示（scale 1.15）

#### 入力フィールド
- クライアント名（テキスト）
- 案件名（テキスト）
- サイト種別（セレクト）：コーポレートサイト、ランディングページ、ECサイト、ブログ・メディア、ポートフォリオ、採用サイト、その他
- 納品希望日（テキスト）
- 案件概要・提案の背景（テキストエリア）
- 制作方針・目的（テキストエリア）
- 実装機能（トグルボタン群、複数選択）：17項目
- 自社名（テキスト）
- 自社URL（テキスト）
- 自社ロゴ（画像アップロード）：
  - ファイル選択でBase64に変換して保存
  - プレビューの表紙とフッターに表示
  - ×ボタンで削除可能
- 備考・特記事項（テキストエリア）

---

### 2. ページ構成タブ

- ページを追加・削除・編集
- 各ページに子ページ（サブページ）を追加可能
- 番号付きリスト表示（1., 2., ...）
- 子ページは「└」で階層を視覚化
- プレビューでツリー型サイトマップを自動生成：
  - TOPノード → 横並びで親ページ → 縦に子ページ
  - テーマカラーのボーダーとバックグラウンドで表示

---

### 3. 見積もりタブ（複数プラン対応）

#### プラン管理
- プランをタブ形式で切り替え
- 「＋」ボタンで新規プラン追加
- 各プランに設定可能：
  - プラン名（編集可能）
  - おすすめフラグ（★マーク、トグル）
  - 複製（現在のプランをコピー）
  - 削除（最低1プランは必要）

#### 見積もり項目
- 項目ごとの入力：項目名、単位（式/ページ/点/時間/月）、数量、単価
- 項目の追加・削除
- プリセットからの追加（サイドドロワー）：
  - 8カテゴリ：ディレクション、デザイン、コーディング、WordPress、フォーム・機能、SEO・集客、セキュリティ、テスト
  - タップで即追加、追加済み表示（1.2秒後にリセット）
  - ドロワーは外側クリックで閉じる

#### 割引・値引き
- 割引タイプ選択：なし / ％割引 / 金額値引き
- 割引値と割引名の入力
- 小計から割引を引いた金額に消費税10%を計算

#### 合計表示
- 小計、割引額（赤字）、消費税、合計（税込）をリアルタイム計算

#### 金額非表示モード
- ヘッダーのトグルでON/OFF
- ONの場合、プレビュー・PDFの見積もりセクションは金額を非表示にし、作業項目名のタグだけ表示
- 「お見積もり金額につきましては、別途ご案内させていただきます」のメッセージ表示

---

### 4. スケジュールタブ

#### 制作開始日
- 日付入力（date picker）、任意
- 入力するとガントチャートに実際の日付を表示
- クリアボタンで解除可能

#### フェーズ管理
- 各フェーズ：フェーズ名、週数（整数）、追加日数（0〜6日）
- 期間表示：「2週3日（17日間）」
- 開始日ありの場合、各行に日付範囲を表示（例：4/12〜4/29）
- フェーズの追加・削除

#### ガントチャート
- 全体期間を表示（例：「8週2日」or「2026年4月12日〜2026年6月7日（8週2日）」）
- 各バーの幅は日数ベースで正確に比例
- テーマカラーのグラデーション
- 開始日あり：バー内に「4/12–4/29」、なし：「2週3日」

---

### 5. 契約条件タブ

#### プリセット選択（各カテゴリ1つ選択、ラジオボタン式）

**お支払い条件（5択）：**
- 着手金50%/納品時50%
- 着手金30%/中間30%/納品時40%
- 納品後一括
- 月額分割
- ココナラ決済

**修正回数（3択）：**
- デザイン2回・コーディング1回
- 各工程3回まで
- 期間内無制限

**著作権（2択）：**
- クライアントに譲渡
- 制作者帰属（使用権付与）

**納品形式（3択）：**
- サーバーアップロード
- データ一式納品
- アップロード＋データ納品

**その他の注意事項（複数選択、チェックボックス式）：**
- 対応ブラウザ
- サーバー費用別途
- 素材ご提供
- 納品後保守
- キャンセルポリシー

---

### 6. プレビュータブ

フルサイズの提案書プレビューを表示。以下のセクションで構成：

1. **表紙** — グラデーション背景（テーマカラー）、タイトル、クライアント名、日付、自社ロゴ・社名
2. **提案概要** — 案件概要と制作方針
3. **制作概要** — サイト種別、総ページ数、納品希望日、制作期間
4. **制作フロー** — ステップ番号付き円形フロー図（矢印接続）、開始日ありなら各ステップに日付表示
5. **実装機能** — タグ形式で一覧
6. **サイトマップ** — ツリー図
7. **お見積もり** — 1プラン：詳細テーブル / 複数プラン：カード比較表示（おすすめプランに★バッジ） / 金額非表示モード：項目タグのみ
8. **制作スケジュール** — ガントチャート（日付 or 週数）
9. **契約条件** — 左ボーダー付きブロック表示
10. **備考**
11. **フッター** — ロゴ、社名、URL、有効期限

## データ保存

### ブラウザストレージ（window.storage API）
- キー: `proposal-projects`
- 最大20件保存
- 各案件データ: `{ id, name, data (form全体), savedAt }`

### 保存・読み込みモーダル
- 「現在の内容を保存」ボタン
- 保存済み案件一覧（日時付き）
- 各案件に「読込」「複製」「削除」ボタン
- モーダルは外側クリックで閉じる

## PDF出力仕様

- プレビューのHTMLを新規ウィンドウにコピー
- Noto Sans JPフォントを読み込み
- ブラウザの印刷ダイアログから「PDFに保存」で出力
- 金額非表示モードの状態がそのまま反映

## UI/UXルール

- すべてのモーダルは外側（オーバーレイ）クリックで閉じる
- プリセットドロワーは右からスライドイン、左側の余白クリックで閉じる
- 実装機能のトグルボタンはON時にチェックアイコン表示
- 入力変更は即座にライブプレビューに反映
- テーマカラー変更はアプリ全体に即座に反映
- 見積もりプリセット追加時は「✓ 済」表示が1.2秒間表示
