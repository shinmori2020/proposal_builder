# 作業ログ

## 確立した共通ルール

### テーマカラーシステム
- 18色を色相順に配置（クリムゾン→チャコール）
- `shades(hex, 7)` でプライマリから7段階グラデーション自動生成
- テーマ変更はアプリ全体（ヘッダー、タブ、ボタン、プレビュー、ガントチャート、フロー図）に即時反映

### モーダル共通仕様
- オーバーレイ（背景暗転）クリックで閉じる
- 内側のコンテンツは `e.stopPropagation()` でクリック伝播を防止
- 対象: テンプレート選択、保存/読み込み、プリセットドロワー

### スケジュール計算
- 各フェーズは「週 + 追加日数（0〜6日）」で管理
- 内部的には日数ベースで計算: `totalDays = weeks * 7 + extraDays`
- 表示は「2週3日（17日間）」形式
- 開始日入力時は実日付に変換して表示

### 見積もり計算
- 小計 = Σ(数量 × 単価)
- 割引額 = ％の場合: 小計 × 割引率 / 金額の場合: そのまま
- 消費税 = (小計 - 割引額) × 10%
- 合計 = 小計 - 割引額 + 消費税
